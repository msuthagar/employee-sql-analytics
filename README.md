# Employee Management & HR Analytics (SQL Project)

## Overview
This project is a simple employee management and HR analytics system built using MySQL and SQL.
It analyzes employee salary, departments, and basic HR data.

## Technologies
- MySQL
- SQL

## Files
- `01_create_tables.sql` → Creates the database and tables.
- `02_insert_data.sql` → Inserts sample data into the tables.

## How to Run
1. Open MySQL Workbench.
2. Run `01_create_tables.sql`.
3. Run `02_insert_data.sql`.
4. Run your own SELECT queries to analyze employees, salaries, and departments.
